# GeoSight — Automated Land Cover Change Detection

**Module:** GIS Programming  
**Author:** Justine Shimpiku · 223065560  
**Institution:** Namibia University of Science and Technology  
**Study Area:** Rundu, Kavango East, Namibia

---

GeoSight is a desktop application that automates land cover classification and change detection using Landsat satellite imagery and a Random Forest machine learning classifier.
